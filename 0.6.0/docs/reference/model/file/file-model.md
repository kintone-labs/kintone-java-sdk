# File Model

Holds file information obtained from kintone.

## Methods

### getContentType()

Get the content type of the file.

**Parameter**

(none)

**Return**

String

### getFileKey()

Get the file key of the uploaded file.

**Parameter**

(none)

**Return**

String

### getName()

Get the name of the uploaded file.

**Parameter**

(none)

**Return**

String

### getSize()

Get the size of the file.

**Parameter**

(none)

**Return**

String
